Conf Cloud 帮助

主持人可以在会议管理界面创建会议，企业用户以"主持人"或者"普通与会者"的身份加入会议，在会议客户端会出现三种角色：主持人、主讲人和 普通与会者。主持人控制整个会议，主讲人对会议中的数据有操作权限，普通与会者参与整个会议过程。

修改基本信息

当企业用户的电话号码或者邮箱地址，密码变更时，企业用户可以修改个人基本信息。

1.企业用户进入会议管理界面。

2.选择"个人设置"。

3.企业用户可以根据实际情况修改用户信息，修改完成后单击"保存"。 系统提示"修改成员信息成功"。

4.单击"确定"，完成用户基本信息修改。

说明:会议安全号，通过会议通话邮件中可以查找到或主持人直接告诉与会者

联系人

添加联系人

创建常用的联系人，在会议邀请时，可以将此联系人名单添加到参会人名单中。

手动添加联系人 1.选择"联系人"。 2.单击"添加"，设置界面参数

从企业通讯录中添加联系人 选择"联系人 > 批量添加联系人"。 单击"从企业通讯录中添加"。 在显示的界面中选择一个或多个联系人。 单击"添加"。 添加联系人成功。

修改联系人

当联系人手机或者邮件地址变更时，企业用户可以修改联系人的信息。 以联系人为frank为例，介绍如何修改联系人。

1.选择"联系人"。

2.选中用户为frank。单击"修改"，修改联系人信息。

3.单击"完成"。 完成联系人信息修改。

删除联系人

当联系人不存在时，企业用户就可以删除该联系人。 以联系人frank为例，介绍如何删除联系人。

1.选择"联系人"。

2.选中用户frank,点击操作中的删除。

3.单击"确定"。 删除联系人成功。

会议开始后邀请其他用户加入会议

会议开始后主持人可以邀请其他用户加入会议。

会议控制

会议客户端支持在会议中切换会议界面显示模式，以满足您不同的会议需求。 在会议中，会议模式不同，主持人和其他与会者设置界面布局的权限也不同：
"主持人模式"：只有主持人可以设置界面布局。
"自由模式"：所有与会者都可以设置界面布局。
设置界面布局步骤如下：

1.在工具栏中，单击，选择"界面布局"。

2.选择"普通显示器模式"或"16:9宽屏显示器模式"，系统显示界面如图所示

录制

在会议中，主持人可以录制会议、设置其他与会者录制会议的权限。

1.在工具栏中，单击，选择"录制"。

2.单击，选择录制文件保存路径。保存的录制文件为"*.wmv"格式的视频文件。

说明： 如果只想录制会议中的声音，不需要录制会议界面上的操作，则可以开始录制前，选择"只录音频"。保存的录制文件为"*.wma"格式的音频文件。
在保存文件时，如果您没有选择保存路径，则系统将文件保存在指定的路径下。

3.单击"保存"。

系统开始录制会议，录制区显示录制状态，如图所示。

录制区中按钮的功能如下：

：暂停录制。 ：恢复录制。 ：停止录制。

4.单击，停止录制。

说明： 如果在录制过程中会议结束，则录制的文件会被保存在指定路径中，会议的结束时间是录制的截止时间。

会场静音和取消静音

为了保证会议中主讲人的声音不被干扰，为与会者创造一个良好的会议环境，主持人在会议中可以选择全场静音，也可以随时开启或者关闭会场任意与会者的麦克风。

会议中麦克风的状态有以下两种：

全场静音

1.在与会者列表工具栏中，单击。主持人设置全场静音后，只允许主持人和主讲人发言，所有参与者的麦克风被关闭，图标变为

说明：在会议中，如果需要参与者参与讨论时，可以取消全场静音，允许与会者发言。您可以单击与会者列表工具栏中的，开启全场参与者的麦克风。

开启摄像头

参与投票

主讲人可以根据会议讨论的内容、与会者的意见，通过问卷调查的方式得出结论。

在工具栏中，单击，选择"问卷调查"。

在如图1所示面板，新建问卷调查的内容。

在新建问卷调查内容时，您可以进行"问题类型"设置、"添加选项"、"添加问题"等操作。
在问卷调查内容新建完成后，您可以单击"保存问卷"，将当前的问卷内容保存到本地文件夹，文件格式为"*.qes"或者"*.htm"。后续可以通过"导入问卷"的方式继续使用。

说明： 在保存问卷时，如果您没有选择保存路径，则系统将问卷保存在指定的路径下。

说明： 在问卷调查面板，您可以单击，新增问卷；单击，删除问卷问题。

设置问卷调查

在问卷调查开始前，您可以设置问卷调查时间、问卷形式和问卷结果的共享权限。

1.在如图1所示面板，单击"设置"。 系统显示"问卷设置"界面。

2.根据实际需要设置如下内容：

问卷时间：完成问卷调查的时间。系统默认为5分钟。
问卷形式：设置匿名或记名问卷调查。系统默认为匿名问卷调查。
结果共享：设置投票结果的共享权限。系统默认为不共享。

开始问卷调查

在问卷调查面板，单击"发起问卷调查"，开始问卷调查。

说明： 主讲人也可以单击"导入问卷"，从本地文件夹中选择问卷发起问卷调查。系统只支持导入在客户端设置的问卷。

结束问卷调查

问卷调查开始后，主讲人可以单击"停止问卷调查"，结束问卷调查。

说明： 如果主讲人没有单击"停止问卷调查"，则系统会在到达设置的问卷时间时自动停止问卷调查。没有单击"提交"的与会者，认为没有参与投票。

查看问卷调查的结果

问卷结束后，主讲人可以直接查看问卷调查结果，也可以保存问卷调查结果。
